# Desenvolvimento do Dashboard Profissional - Bar

## Início da Conversa
**Data:** 10 de Maio de 2026

**Objetivo:** Replicar as telas do Home Assistant (Home, Salão, Andar 1, Bar) em dashboards externos profissionais, começando pelo **Bar**.

**Contexto:** O usuário deseja uma interface mais profissional e "limpa" para uso externo, sem alterar o dashboard original do Home Assistant. O dashboard do Bar atualmente usa uma planta 3D interativa.

**Plano de Ação:**
1. Analisar as entidades usadas no dashboard do Bar.
2. Criar um Web App (Vite + React) com design premium.
3. Integrar com a API do Home Assistant usando o Token já gerado.
4. Implementar o dashboard do Bar como o primeiro módulo.
